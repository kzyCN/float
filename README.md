# float

[![](https://img.shields.io/badge/blog-@fsyume-red.svg?style=flat-square)](https://www.fsyume.com)![](https://img.shields.io/badge/version-v1.0-blue?style=flat-square)![GitHub](https://img.shields.io/github/license/kzycn/float?style=flat-square)

> 由SpringBoot构建的个人文章管理系统

本系统为前后端分离项目，前端使用Vue全家桶，组件库为Element UI，后端为Springboot+MyBatis+jwt

主要更新地点为Github，国内镜像Gitee会有延迟

Github地址：https://github.com/kzycn/float

Gitee地址：https://gitee.com/kzycn/float
